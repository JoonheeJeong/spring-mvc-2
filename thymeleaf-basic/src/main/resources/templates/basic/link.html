링크

  • th:href="@{/hello}"
     => basic url
  • th:href="@{/hello(param1=${param1}, param2=${param2})}"
     => query parameter
  • th:href="@{/hello/{param1}/{param2}(param1=${param1}, param2=${param2})}"
     => path variable
  • th:href="@{/hello/{param1}(param1=${param1}, param2=${param2})}"
     => path variable & query parameter
